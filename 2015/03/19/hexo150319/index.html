使用Hexo建立自己的博客

此文为小女子第一篇技术博客，记录了建立此博客的艰辛过程。

关键词:

  • node.js
  • github
  • hexo

Step 1 ：安装node.js

  • nodeJs 官网下载地址:https://nodejs.org/download/

    本人win8 64位电脑。下载版本为
    Windows Installer(.msi) 64bit
  • 点击安装

Step 2 :下载git

  • 下载安装git.exe，可百度搜索该软件，参考下载地址
  • 在任意文件夹中或桌面，右击鼠标，看到 Git Bash 即安装成功

Step 3 :安装Hexo

  • 在桌面鼠标右击，点击 Git Bash，在shell中敲入命令:

    npm install -g hexo
  • 在D盘或者E盘打开Git Bash 依次执行

    $ hexo init blog
    $ cd blog
    $ npm install
    $ hexo server
    此时，多出文件夹 blog文件夹
    文件说明：
    参照文章

在浏览器中输入http://localhost:4000/ 即可看到效果，此效果为默认效果，若想更换主题，点击此处,选取喜欢的主题，获取主题的git地址，在blog/theme文件夹下执行：$ git clone 主题地址
打开 blog/_config.yml,找到theme,把默认主题landscape改成你的主题名
重启服务(ctrl C,hexo server)，浏览器中回车刷新页面，新主题博客出现^^

Step 3 :推送至GitHub

  • 在github上注册账号。
  • 百度git ssh生成公钥，参考
  • 注意: https://github.com/settings/emails一定要验证邮箱，一般在注册的时候，就会让验证，点击发送到你邮箱的一个链接
  • 在settings里，找到 ssh keys 一项，添加ssh key.参考

  • 新建仓库，new repository

    repository name 你的用户名.github.io,例如我的 wqpage.github.io

  • 修改_config.yml 最后deploy
    deploy:

    type: git

    repository: https://github.com/wqpage/ wqpage.github.io.git

    branch: master

    *在blog目录下，git bash，输入命令:

    $ npm install hexo-deployer-git —save

    $ hexo generate

    $ hexo deploy

完成，在网页上输入http://你的账户名.github.io/ 本例参考 http://wqpage.github.io/
